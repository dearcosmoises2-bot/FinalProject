Landfill e-Forms Application

The landfill e-forms project converts the City of Los Angeles, Department of Sanitation's methane emission auditing system from paper to fully electronic in order to improve efficiency, reduce human error, and speed up report production. This project consists of two applications: a mobile application and a web application for desktop use. The mobile application runs under the Android Operating System and will be used by inspectors in the field. The web application is developed with an Angular 4 front-end and a Java/Spring back-end. The user is able to upload/download data to and from the web application through a hardwired sync when connected to a mobile device running the mobile application. The web application serves as the main source for data entry, data search, and generating reports.
